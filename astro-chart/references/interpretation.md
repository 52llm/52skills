# 占星解读框架（agent 用）

排盘数据由 `scripts/astro_chart` 产出；本文是把数据变成解读的工作方法。
按顺序走，每个论断挂上盘面证据（某行星某座某宫、某相位 orb 多少）。

## 本命盘解读流程

1. **三巨头**：太阳（核心自我/意志）、月亮（情绪需求/安全感）、上升（外在姿态/
   人生切入点）。三者星座的元素组合先定基调。
2. **元素/模式失衡**（`elements/modes`）：某元素 ≥5 为过强、0 为缺失——
   火=行动力，土=务实，风=思维社交，水=感受共情；基本=开创，固定=坚持，变动=适应。
   缺失元素是成长课题，不是缺陷。
3. **逐行星读配置**：行星（功能）+ 星座（风格）+ 宫位（领域）。个人行星
   （日月水金火）权重最高；木土讲课题；三王星按宫位读、按世代弱化星座意义。
4. **紧密相位优先**：orb <2° 的相位主导人格张力；日月、日升相位再加权。
   合=融合放大，冲=拉扯与投射（常经由关系呈现），刑=摩擦推动行动，
   拱=天赋顺畅（易被忽视），六合=机会需主动。硬相位≠凶，是能量出口。
5. **宫位重点**：星群（≥3 星同宫/同座）所在领域是人生主战场；角宫
   （1/4/7/10）行星外显强。宫头星座+宫内行星+宫主星去向三层读。
6. **整合**：找 2–3 条反复出现的主题（如"水星天蝎+八宫星群+冥王刑月=洞察与
   控制议题"），围绕主题组织报告，不要逐条罗列辞典。

| 水星 | 思维、沟通 | | 3 | 学习、手足、短途 |
| 金星 | 爱、审美、吸引 | | 4 | 家庭、根基 |
| 火星 | 行动、欲望、怒 | | 5 | 创造、恋爱、子女 |
| 木星 | 扩张、机遇、信念 | | 6 | 工作日常、健康 |
| 土星 | 限制、责任、成熟 | | 7 | 伴侣、合作、公开敌人 |

星座关键词按元素×模式推：白羊=火+基本（直接开创）……双鱼=水+变动（共情消融）。

## 行运（transits）读法

1. 只认**外行星对个人行星/四轴**的行运（木土天海冥 → 日月水金火 ASC MC）；
   行运月亮水星金星太阳变化太快，只作日常氛围。
2. 优先级：orb 越紧越当令；合冲刑 > 拱六合；行运土星/冥王星主结构性课题，
   木星主扩张窗口，天王星主突变，海王星主迷雾期。
3. 经典周期主动提：土星回归（29 岁、58 岁前后，transit 土合本命土）、
   土星对冲（14/44 岁）、天王对冲（42 岁前后）、木星回归（12 年机遇轮）。
   行运报告先看 `transit_in_natal_houses`（外行星走到哪个宫=哪个领域被激活），
   再叠相位。
4. 逆行：本命逆行=该功能内化；行运水逆只影响水星主题（沟通/出行/合同），
   幅度务实描述，别妖魔化。

## 合盘（synastry）读法

优先看：①日月/日升跨盘相位（核心契合）；②金火相位（吸引力）；
③月亮相位（生活舒适度）；④土星相位（承诺与压力并存）；⑤外行星压个人行星
（天=不稳定，海=理想化，冥=强度与控制）。硬相位是"有火花但有功课"，
全无相位比多硬相位更难维系。报告分"吸引点/摩擦点/长期课题"三段。

## 表达纪律

- 用"倾向/主题/课题"而非"必然发生"；健康、生死、婚姻存续等敏感话题只给
  倾向性提醒，建议以现实判断为准。
- 出生时间不确定时（warnings 有提示），明确说明宫位/上升不可靠，
  月亮换座时给两种可能各一句。
- `near_sign_boundary` 的行星要注明"贴近星座边界，以精确出生时间为准"。
- 世代行星（天海冥）的星座是同代人共有的，个人化解读要靠宫位与相位，
  不要拿"冥王天蝎座"当个人特质。
