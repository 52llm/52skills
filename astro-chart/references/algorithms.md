# astro-chart 算法与精度

实现全部在 `scripts/astro_chart.py`，纯标准库。所有黄经均为**当日分点**
（equinox of date）的地心视位置——占星星座即按此划分。

## 位置计算

- **时间**：输入经 zoneinfo 转 UTC → 儒略日 → ΔT（Espenak–Meeus 分段多项式）→ TT。
- **太阳**：VSOP87D 地球日心黄经截断级数（Meeus 表 32.a，L0 64 项起），
  +180° 转地心，加 FK5 修正、章动主 4 项、光行差 −20.4898″/R。精度 ~1″。
- **月亮**：Meeus 第 47 章 ELP 截断（黄经/距离 60 项 + A1/A2 附加项，黄纬 30 项 +
  附加项），含 E 偏心率因子与章动。精度 ~10–15″（月亮每分钟走 33″，
  对星座/宫位判断绰绰有余）。
- **行星（水星–冥王星）**：JPL《Approximate Positions of the Planets》
  Table 1 开普勒根数 + 世纪变率（**适用 1885/1800–2050**），牛顿法解开普勒方程，
  轨道面→J2000 黄道坐标。地球取 EMB 再用月亮位置修正到地心（消 ~20″ 偏差）。
  地心向量差后：光行时迭代 2 轮 → 岁差（+5029.0966″T+1.11113″T²，J2000→当日分点）
  → 章动 → 周年光行差（−20.4898″·cos(☉−λ)/cosβ）。
  实测 2024 年木/土/天/海/冥与权威星历差 ≤6′（土星留驻位置差 5′），
  内行星类似量级。**冥王星在根数适用段内同样角分级**，段外退化明显。
- **北交点**：平交点多项式；默认加 Meeus 真交点摄动修正（±1.5° 振荡），
  `--mean-node` 切平交点。
- **逆行/速度**：±0.5 天黄经差分；相位入相/出相由相对速度外推判断。

## 上升、天顶与宫位

- **恒星时**：GMST（Meeus 12.4）+ 章动·cosε 得视恒星时；RAMC = LST + 东经。
- **MC**：λ = atan2(sin RAMC, cos RAMC·cosε)（闭式）。
- **ASC**：不用闭式公式（象限易错），改为对黄道逐点求
  地平高度 sinφ·sinδ + cosφ·cosδ·cosH 的零点（2° 网格 + 60 轮二分），
  取时角 H<0（东方）的根。回归测试断言 |alt|<1e-6 且在东方。
- **Placidus**：标准迭代。宫头 11/12/2/3 满足
  RA = RAMC + {30,60,120,150}° + AD·{1/3,2/3,2/3,1/3}，AD = asin(tanφ·tanδ)；
  其余宫头对冲。|φ|>66° 或迭代中 |tanφ·tanδ|≥1（拱极）时判失效，自动退整宫制
  并在 warnings 说明。回归测试用随机 (RAMC, φ) 验证宫头满足定义方程（<0.01°）。
- **整宫/等宫**：ASC 星座 0° 起每 30°；ASC 度数起每 30°。
- 行星落宫按宫头区间（黄道顺行方向）。

## 相位

五大相位 0/60/90/120/180。默认容许度：本命 {合8 六合5 刑7 拱8 冲8}，
日月参与 +1°；行运 {3,2,3,3,3}；合盘 {6,4,5,6,6}；`--orb-scale` 整体缩放。
按 orb 从紧到松排序输出，只保留每对行星最匹配的一个相位。

## 已知限制

- 小行星（谷神星等）与凯龙星需要数值星历，未内置；提及时如实告知不支持。
- 行星根数段外（<1800 或 >2050）外行星误差可达度级，输出会带警告；
  太阳/月亮不受影响。
- 恒星（恒星合轴）、汉堡学派虚点、日返/月返盘未实现；月返/日返可用
  transits 近似（找太阳回归时刻需外部迭代）。
- 极圈内（|φ|>66°）Placidus 数学上无定义，自动退整宫。

## 校验命令

```bash
python3 - <<'EOF'
import sys; sys.path.insert(0, "scripts"); import astro_chart as ac
jde = ac.jd_ut_to_jde(ac.gregorian_to_jd(2024, 3, 20 + 3.1056/24))
print("春分太阳黄经(应≈0):", round(ac.sun_apparent_lon(jde) % 360, 4))
EOF
```

改动级数或宫位代码后，跑 SKILL.md「校验与精度」列的全部锚点。
