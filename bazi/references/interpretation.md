# 八字解读框架（agent 用）

排盘数据由 `scripts/bazi` 产出；本文是把数据变成解读的工作方法。按顺序走，
每一步引用盘面证据（干支、十神、月令），不要泛泛而谈。

## 解读流程

1. **认日主**：`day_master`（日干五行阴阳）是"我"。
2. **定旺衰**：看三件事——
   - 得令：日主五行在月支的状态（下表）；月令权重最大。
   - 得地：四支藏干中有无日主的根（同五行本气根 > 中余气根）。
   - 得势：天干比劫印星多寡。
   - 三者多助为**身强**，多耗泄克为**身弱**；介于其间称中和。`wuxing` 的两组计数
     只是参考，旺衰以月令为纲。
3. **取用神**（扶抑为主）：身弱→用印比（生扶），忌财官食伤过重；
   身强→用财官食伤（耗泄制），忌印比再帮。极端偏枯考虑从格（从强/从弱），
   从格宁缺勿滥。冬夏生人兼看**调候**：冬水寒需丙火，夏火燥需壬癸水等。
   两神相战考虑**通关**（如金木相战用水通关）。
4. **看格局**：月支本气透干即取格（正官格、七杀格、正偏财格、食神格、伤官格、
   正偏印格；比劫当令看建禄/羊刃）。格局定性 + 用神定量。
5. **读大运流年**：大运干支与原局的生克合冲 > 流年。运支冲月支/日支、流年干支
   合冲用神忌神，是重点年份。用神得助的大运为顺境，忌神当道为逆境。
6. **落宫位**：年柱=祖上/早年，月柱=父母兄弟/青年，日支=配偶/中年，时柱=子女/晚年。
   神煞（桃花驿马贵人等）只作点缀性佐证，不单独定吉凶。

## 十神含义速查

| 十神 | 关键词（人格面） | 事象面 |
|---|---|---|
| 比肩 | 自立、同辈、固执 | 合伙、竞争、分财 |
| 劫财 | 行动力、冲动、争夺 | 破财、义气之交 |
| 食神 | 才艺、温和、口福 | 表达、子女(女命)、生财之源 |
| 伤官 | 聪明外露、叛逆、创造 | 技艺成名、克官（体制冲突）|
| 偏财 | 灵活、慷慨、机会财 | 流动收入、父亲、异性缘(男) |
| 正财 | 务实、节俭、稳定财 | 工资、妻星(男命) |
| 七杀 | 魄力、压力、竞争心 | 权柄、小人、官非；制化得宜为大器 |
| 正官 | 自律、名誉、责任 | 职位、体制、夫星(女命) |
| 偏印 | 敏感、独门技艺、疑心 | 冷门学问、继母；夺食为忌 |
| 正印 | 仁慈、学业、庇护 | 学历、母亲、贵人、名声 |

强弱搭配示例：伤官配印（才华+约束=成就）、食神制杀（压力转权力）、
杀印相生（逆境出功名）、财旺生官（富而后贵）、比劫夺财（破财、感情竞争）。

## 合冲刑害的用法

- 天干五合：合而能化看月令；合羁绊用神最要紧（如日干被合）。
- 六冲：冲提纲（月支）动荡大；子午卯酉冲主人事变动，寅申巳亥冲主奔波，
  辰戌丑未冲开库（视喜忌）。
- 三合三会：会成的五行力量倍增，直接改写旺衰判断。
- 刑/害：作辅助信号（刑主纠葛/官非/健康，害主暗损），不宜过度演绎。

## 大运流年报告模板

对每步大运给三行：①干支+十神+与原局最关键的一个作用（合/冲/生克哪个字）；
②该十年的主题词（事业/感情/健康侧重）；③注意年份（流年与大运或原局形成
冲提纲、伏吟反吟、岁运并临的年）。用户问具体年份时对流年做同样三行。

## 表达纪律

- 输出用"倾向/偏向/提示"而非"必然/注定"；健康、寿夭、婚灾等敏感话题只给
  倾向性提醒并建议以现实判断为准。
- 每个论断挂上盘面证据（"因为月令酉金当令而日主甲木无根"），可被追问验证。
- 流派分歧处（子时换日、真太阳时、神煞取舍）如实说明采用的默认并提供开关。
- 排盘警告（⚠ 距交节很近等）必须转述给用户。
