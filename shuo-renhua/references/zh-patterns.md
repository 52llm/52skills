# zh-patterns — 中文 AI 腔模式（九组）

用法：按 registers.md 的语体激活表选组扫描。每条 = 关键词/形态 → 触发条件 → 改法。
**密度原则贯穿全部**：单独出现不算，短段堆叠、脱离内容、空转才算。⚙ 标记 = `scripts/shuo_renhua check` 能机械抓到。

## A. 内容拔高与套路（公文/应试不触发）

- **过度拔高意义**：具有重要意义/标志着里程碑/谱写新篇章/树立了标杆 → 删拔高句，落回具体做了什么。
- **空洞强调**：值得一提的是/不可否认/毋庸置疑 → 直接说事。
- **⚙时代背景开场**：随着…的不断发展/在…的大背景下/当今时代 → 砍掉开场白，第一句进正题。
- **尾巴式升华**：…体现了/彰显了/折射出…的深层逻辑 → 结尾落在事实、后果或下一步。
- **模糊权威归因**：专家表示/业内人士认为/据悉/相关研究显示（无出处）→ 有来源给来源，没来源改成自己的判断。品牌文案不触发（品牌自己就是权威）。
- **公式化"挑战与展望"**：尽管面临…但我们相信…必将迎来 → 写具体的困难和具体的计划。

## B. 高频词与词汇

- **⚙AI 高频词聚簇**：赋能/助力/打造/护航/抓手/闭环/生态/底层逻辑/核心竞争力/高质量发展/全方位/多维度/多层次/全链路/一站式/见证/根植于/至关重要。**≥3 个同现即触发**。
- **⚙华丽意象堆砌**：璀璨/熠熠生辉/画卷/华章/乐章/扬帆起航/乘风破浪/砥砺奋进 → 全删，写下一季度实际要做的事。
- **同义词轮换**：<100 字内对同一指称换 ≥3 个说法（主角→主人公→核心人物→英雄）→ 收敛为一个词+代词。
- **万能动词**：换任何话题都能用的动词（激发/实现/推动/发挥/开展）→ 换只有这件事能用的动作（逼香/跑通/说定/管用）。
- **的的不休**：短句连用 ≥3 个"的" → 拆句或删定语。
- **进行+V**（语体敏感）：进行讨论→讨论。学术/公文里是规范表达不改，日常密集堆叠才改。
- **性/化后缀堆叠**（语体敏感）：白名单术语不改（可持续性/鲁棒性/数字化/自动化），临时造的"抓手性/闭环化"改。
- **中英强行括注**：算力(compute)/推理(inference) 冗余并列 → 首次引入可括注一次，后文只用一形。独立缩写（MCP/API）按白名单保留。
- **⚙空枢纽**：值得注意的是/需要注意的是/真正的问题是/核心在于/说到底/归根结底 密集出现 → 删枢纽直接说。"说到底"偶发于口语不误杀。
- **假坦诚开场**：讲真/说实话 当钩子 + 后接套话推销 → 删。接不确定反应（"说实话我也不知道"）是毛边，保留。
- **列表预告**："这里有几个要点""下面从三个方面" → 直接列，不预告。

## C. 句壳与修辞

- **"不仅…更…"空洞并列** 与 **否定对举定义句**（这不是 X，而是 Y / 既不是 X 也不是 Y）：工整对称 + X/Y 是抽象桶词 + 不落到具体 → 触发；用来剥离真实误解、落到具体真相 → 保留。
- **虚假范围**：从 X 到 Y 不在同一刻度（"从星辰大海到内心宇宙"）→ 改成并列或删。"从 NBA 到欧冠"是真并列，不触发。
- **四字成语机械堆叠**：凝心聚力、砥砺前行、开拓创新 ≥3 连用 → 留一个或全删。公文/应试/有意排布的特稿不触发。
- **排比模板**：首先/其次/最后 硬套、三段等长排比 → 破对称：删一句、把一句扩成具体例子、留一句白描。品牌文案紧扣情境的排比是利器，不触发。
- **总-分-总僵化**：四条同时成立才触发（抽象开头 + 恰好 3 分论 + 空总结 + 无新细节）。
- **⚙过度加粗与 emoji 装饰**：每两句一个 **加粗**、🚀💡✅ 列点 → 全删。加粗一篇 ≤3 处且只标真正的关键。
- **破折号滥用**：非杂文/特稿/品牌语体里 500 字 ≥2 处不承担功能的 `——` → 换逗号/冒号/括号/另起一句。中文真人破折号基线接近 0（输入成本高、有零成本替代），这是它信号可靠的原因；但**中文破折号不是英文式全域禁令**，杂文/特稿/品牌节奏里的功能性使用保留。

## D. 助手残留（所有语体全触发）

- **⚙客服腔**：希望对您有帮助/如有疑问请告诉我/祝您生活愉快/很高兴为您服务。
- **谄媚**：您问得非常好！/您说得完全正确！
- **⚙知识截止**：截至我的知识更新/作为一个 AI 语言模型。
- **⚙空洞积极结尾**：未来可期/前景光明/大有可为/让我们拭目以待 → 结尾落在计划或事实。
- **"让我们一起…"号召**：让我们一起开启精彩新篇章 → 删。品牌祈使（Just Do It/上）不触发。
- **第二人称泛化**：对读者说话却写"用户/大家/各位"当主语 → 改"你"。学术里指第三方的"用户"不触发。

## E. 填充与模糊

- **冗余书面化**：通过…的方式/在…的情况下/由于…的原因 密集 → 压缩。公文里可能是必要精确度。
- **模糊限定连叠**：在一定程度上/某种意义上/可能/也许 连用 → 一句话最多一个限定词。
- **逻辑连接词空转**：然而/此外/因此/综上所述 短段 ≥3 个且删掉不影响逻辑 → 删。学术里**空转的**同样删，承担真实逻辑的保留。

## F. 翻译腔

- **英式语序**：介词短语开头过长/状语后置（"我们讨论了 X 在会议上"）/定语从句堆成长"的"字链/主语埋在句中 → 按中文语序归位：话题在前、状语在动词前、修饰短。
- **"作为一个 X"**（语体敏感）：日常里硬套刺眼 → "做产品经理这几年，我觉得"。学术/社论庄重用法合法。
- **被字密度**：中文被动天然稀少，连续"被"句 → 改主动或受事主语句（"这个问题解决了"）。
- **物理动词套心理过程**：接住情绪/击穿认知/这个观点很锋利 → 英文 catch/pierce 直译，换中文说法。
- **"X 很 Y："起手**：逻辑很清晰：/结论很明确： → The logic is clear: 直译骨架，重写。
- **⚙该译不译**：有通行中文说法却留英文（argument/workflow/feature 单蹦）→ 译。白名单（MCP/API/TLDR/embedding/token 等圈内形态）例外，见 registers.md。
- **⚙半角标点混入中文**：中文正文一律全角（，。：；？！）；并列用顿号不用逗号；引出用冒号不用破折号；括号=可跳过的次要补注、破折号=不可跳过的重要补注。代码/URL/版本号内保留半角。

## G. 篇章与节奏

- **句长均质化**：连 3 句 20-30 字全是完整主谓宾（机关枪）→ 打断：合并碎句、拆开长句、插一个短判断。
- **段落均匀**：每段长度几乎一样 → 轻重错开（短拍-中说明-厚段-短落地）。
- **指代不敢省**：专名一段重复 ≥5 次 → 用代词或省略（中文主语可承前省）。品牌文案重复品牌名是记忆强化，不触发。
- **抽象占位例子**：用 A/B/C、"某公司"代替具体实例 → 真人讲经验用真例子；没有真例子就不举。
- **列表默认反射**：连贯叙述被硬拆成 `## 1. 先… ## 2. 然后…` → 还原成散文。技术文档/操作步骤/PRD 不触发。
- **表格滥用**：每行都是完整句、整读像故事的表格 → 还原散文。参数对照/命令表保留。

## H. 平台套路（仅自媒体/社交语体激活；详表见 registers.md 平台节）

- 公众号伪深度：咨询黑话堆砌（颗粒度/心智/第一性原理/范式转移）+ "不是 X 而是 Y"伪洞察 + 升华结尾。诊断：把术语换成大白话看掉不掉信息，不掉=伪深度。
- 小红书：⚙info-block emoji 微模板（✅必点/📍地址/⏰营业时间）、⚙已入土流行语（yyds/绝绝子/栓Q/家人们谁懂/姐妹们快冲）、伪疗愈黑话（把自己重养一遍/能量场/内核稳定）——"正确的废话+排比+情绪平滑，没有具体崩溃瞬间、具体账单"。hashtag 压到 3-5 个、删平台冗余标签。
- **毒性正能量缝合**（单次即判）：负面流行语 + 180° 升华转折（"作为打工人…但正是这份坚持让我成长"）——真人讲累讲完就算了，每次必升华的是 AI。
- 短视频解说腔：注意看，这个男人叫小帅。B 站：时间轴幻觉/每段一个"你可以把 X 理解成 Y"。

## I. 硬证据（单次出现即触发，全语体）

- **⚙Markdown 残留**：`**粗体**`/`# 标题`/`---` 晾在不认 Markdown 的环境（公众号/朋友圈/Word）。
- **⚙编造引用与 AI URL**：虚构 DOI/ISBN；URL 带 `utm_source=chatgpt.com|claude.ai` 尾巴。
- **⚙模板占位符残留**：XX 路/XXX 号/2024 年 X 月/[产品名]/{{变量}}——不修复、不发明具体值，标"[占位符请核对]"或删。
