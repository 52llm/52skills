# components — 组件数值规范与状态八格

产品 UI 优先用可达性组件基元（Base UI / React Aria / Radix；项目已有的优先），**禁手写键盘/焦点行为**，禁在同一交互面里混两套基元。以下是生成式模型最容易做错的数值与行为（精选自 ui-design-brain 60 组件库）。

## 高频组件速查

- **Button**：触达 ≥44×44px（桌面高度 ≥36px）；异步时按钮内转 spinner 并禁用防双击；每个视口区**一个** primary；表单页脚 primary 靠右、secondary 靠左；文案 ≤3 词且桌面端不折行。
- **Card**：内部层级固定 媒体→标题→meta→动作；网格里等高（文字 line-clamp）；阴影或边框**二选一**；代表可导航实体时整卡可点。
- **Modal**：三重关闭（X、取消、Esc）；焦点锁在内部、关闭后还给触发元素；内容要滚动就该换成整页；modal 套 modal 永远是错的。
- **Drawer**：详情从右滑入、导航从左；桌面宽 320-480px、移动全宽；半透明遮罩定焦点。
- **Dropdown/Menu**：7±2 项，更多用子菜单或搜索；贴近视口边缘自动翻向；破坏性动作红色、放最后、分隔线隔开。
- **Toast**：非关键 4-6s 自动消失；新的叠上面；可撤销操作带 Undo 动作；一行文案。
- **Tabs**：2-7 个，多了滚动或收进下拉；内容紧贴 tab 栏无缝隙；窄视口考虑改手风琴。
- **Table**：滚动表头 sticky；数字列右对齐 + tabular-nums；移动端横向滚动而不是砍列；可操作表带全选列；>100 行虚拟化。
- **Combobox/Search**：输入 1-2 字符后出建议；命中文本高亮；无结果给"无结果"文案不留空框；防抖 200-300ms；搜索框带 ⌘K 提示与清除 ×。
- **Avatar**：三档 24-32/40-48/64-80px；降级链 图片→首字母→通用图标；群组叠排 + "+N" 溢出。
- **Checkbox/Radio**：checkbox 对齐 label **首行**不是垂直居中；"全选"用 indeterminate；radio >2 项竖排。
- **Segmented control**：2-5 段等宽。**Slider**：滑块触达 44px。**Tooltip**：hover ~300ms 后出现，hover+长按触发（不点击）；装饰键位用 `<kbd>`。
- **Accordion**：展开收起 150-250ms；FAQ 别默认手风琴（可考虑并排列表/可搜索）。
- **Skeleton**：~200ms 后才出现（防闪烁），形状匹配最终布局；<300ms 的加载不配骨架。**Spinner**：延迟 ~300ms 出现；inline 16px / 按钮 20px / 页面 40px+。
- **Header**：高 56-72px（营销页上限 80px）。**Tree**：每层缩进 16-24px。
- **Empty state**：精心排版 + **一个明确的下一步动作**，不是一句"暂无数据"。

## 状态八格（每个数据面都过一遍）

| 状态 | 契约 |
| --- | --- |
| idle | 默认态即设计过的态 |
| loading | 骨架屏 ~200ms 后出现且几何匹配最终布局；禁全屏大转圈 |
| empty | 指出如何产生第一条数据，给动作按钮 |
| error | 原因 + 一键恢复路径 + （产品 UI）支持 ID；语气直接，禁"Oops!" |
| partial | 缺失字段显示 `—`，永不 null/N/A/0 |
| conflict | 并发编辑/过期数据的提示与合并路径（协作面才需要） |
| offline | 断网可感知、恢复后自动重试（联网面才需要） |
| success | 确认动作完成（toast/内联），文案与按钮同名（Publish → Published） |

复杂状态用状态机思维建模，别用 `isLoading && !error && data` 这种可组合出非法状态的布尔堆。
产品 UI 至少覆盖 idle/loading/error；数据面加 empty/partial；协作与离线面全八格。

## 组件级反模式

彩虹徽章（语义色 ≤4 种）；桌面端汉堡菜单（导航 ≤7 项就平铺）；自动轮播（用户没碰就别动）；placeholder 当 label；正文 <14px（最小 12px 只给密集表格 meta）；全暗侧栏（用 tint 过的中性色）；比例数字用在数字列（用 tabular）；"OR"分割线全大写（用小写 or）；登录页旁整幅饱和品牌大图版。
