# tokens — 数值基准（没有品牌 token 时的缺省，有则以品牌为准）

发出去的是这些具体数字，不是"和谐、呼吸感"之类的形容词。来源以 ui-craft 实测体系为主（见 sources.md）。

## 间距（8 级，8px 网格）

```css
--space-xs: 0.25rem;  /*  4px 标签→输入框、图标→文字 */
--space-sm: 0.5rem;   /*  8px 组内条目之间 */
--space-md: 1rem;     /* 16px 相关组件之间 */
--space-lg: 1.5rem;   /* 24px 兄弟区块之间 */
--space-xl: 2rem;     /* 32px 组件→组件 */
--space-2xl: 3rem;    /* 48px 小节分隔 */
--space-3xl: 4rem;    /* 64px 页面大区 */
--space-4xl: 6rem;    /* 96px hero→正文；营销页大节间距（py-24 起步，画廊感 py-32~48） */
```

- **不变量：组内间距 < 组间间距 < 节间间距**（标签距输入框 4px、同表单输入框相距 16px、表单距下一节 48px+）。节间 ≥ 2× 块间。
- 密集仪表盘变体：只用 4/8/12/16/24，永不 5/7/9/10；单元格 `px-3 py-2`；侧栏 200-240px。
- 光学修正：视觉中心比几何中心高 5-8%；图标字形常需 -1px 微调；顶底 padding 按眼睛调（底部常需略大）。

## 字阶与排印

- 阶梯：`12 / 14 / 16 / 18 / 20 / 24 / 30 / 36 / 48 / 64`（px）。不出现 13/15/17 这类一次性字号（光学修正除外）。
- **tracking 随字号**：≥24px → `-0.02em`（40px+ 可到 `-0.04em`，**下限 -0.04em**，再紧字母就贴在一起了）；正文 14-18px 不动；11-13px 小标签 → `+0.01~0.02em`（这是唯一允许全大写的档位）；全大写 → `+0.05~0.1em`。
- **行高**：标题 1.05-1.2；正文 1.5-1.65；UI 标签 1.3-1.4；**中日韩正文 1.7-1.85**。
- **字重 ≤3**：400 正文、500 UI/标签、600-700 标题。别加载 9 个字重。
- 行长：正文 45-75ch（理想 66，`max-width: 65ch`），移动端 35-60ch；长文正文字号下限 18px。
- 高端实测区间（营销页）：Hero 76-94px/700/lh1.05/-0.04em；节标题 48-56px/700/lh1.1/-0.03em；卡片标题 24-32px/600/lh1.2；正文 16-17px/400/lh1.5。Hero clamp 上限 ≤6rem(96px)——再大是喊叫不是设计。
- 斜体降部（y g j p q）在 display 里会被 `leading-none` 裁掉：至少 `leading-[1.1]` + 底部留 `pb-1`。
- 字体配对按**对比轴**配（衬线+无衬线、几何+人文），不配"相似但不相同"的两个几何无衬线；或干脆一族多字重。
- 数字列/对比数据必须 `font-variant-numeric: tabular-nums`；标题 `text-wrap: balance`，长段 `text-wrap: pretty`。

## 圆角 / 阴影 / z-index

- **圆角随元素变化**（全页统一一个值是"生成感"最强的单一信号）：输入框 4-6px、按钮 8px、卡片 8-16px（随卡片大小）、模态 14px、胶囊 9999px。嵌套时子 ≤ 父：`calc(var(--radius-parent) - var(--padding))`。卡片圆角 >20px 是已知的模型式过度圆角。
- **阴影分层**（环境光+直射光两层），浅色模式：

```css
--shadow-sm: 0 1px 2px rgba(0,0,0,.05);
--shadow-md: 0 4px 6px rgba(0,0,0,.07), 0 1px 3px rgba(0,0,0,.06);
--shadow-lg: 0 10px 15px rgba(0,0,0,.10), 0 4px 6px rgba(0,0,0,.05);
--shadow-xl: 0 20px 25px rgba(0,0,0,.10), 0 8px 10px rgba(0,0,0,.04);
```

  阴影色调向背景色相靠拢，浅底禁纯黑投影。**暗色模式不用投影**：用表面提升（在底色上叠白 6%/8%/12%）+ 1px 亮线。卡片"1px 边框 + 大模糊阴影"二选一，别叠加（ghost-card 是已知模型缺陷）。
- **z-index 语义梯**（禁 999/9999）：dropdown 10 · sticky 20 · modal-backdrop 30 · modal 40 · toast 50 · tooltip 60。

## 时长 / 缓动 token（用法见 motion.md）

```css
--motion-fast: 120ms;   /* hover、tooltip、透明度 */
--motion-base: 200ms;   /* dropdown、toggle、tab */
--motion-medium: 280ms; /* modal、drawer、popover */
--motion-slow: 400ms;   /* 页面转场 */
--ease-out:        cubic-bezier(0.22, 1, 0.36, 1);  /* 入场与大多数 UI */
--ease-in-out:     cubic-bezier(0.65, 0, 0.35, 1);  /* 同层切换 */
--ease-emphasized: cubic-bezier(0.2, 0, 0, 1);      /* hero，一个视口最多一处 */
```

自造 `153ms` 这类零散时长本身就是一个 finding。

## 颜色

- 用 **OKLCH** 写色阶（感知均匀）。中性色带一点品牌色相（chroma +0.005~0.015），不要默认往"暖"或"冷"偏——那是跨项目单一文化的来路。
- 大面积禁纯 `#000`/`#fff`；暗色底 `#0a0a0a~#141414` 档，正文最深观测值 `#18181b`。
- **对比度**：WCAG 正文 ≥4.5:1、大字(≥24px 或粗体 ≥18.5px) ≥3:1、非文字 UI ≥3:1（AAA 正文 7:1）；用 APCA 则 Lc≥75 正文 / Lc≥60 大字 / Lc≥45 非文字。placeholder 同样要 4.5:1。`scripts/ui_taste contrast` 可算。
- 图片描边（唯一不跟色调走的地方）：浅色 `outline: 1px solid rgba(0,0,0,.1)`、暗色 `rgba(255,255,255,.1)`，`outline-offset: -1px`。
- 状态点 6-8px 实心圆，不做胶囊。关键指标强调：底 8% 透明度的强调色 + 数字用全强度。
- **色彩策略四档**（先选档再选色）：克制（中性+一个强调 ≤10%，产品默认）/ 承诺（一个饱和色占 30-60% 面，品牌页默认）/ 全色板（3-4 个具名角色）/ 浸染（表面即颜色，campaign 用）。
- 明暗主题不是默认值：先写一句物理场景（谁、在哪、什么光线、什么心情）逼出答案，写不出就再具体。

## 无品牌时的成套主题（选一套整用，禁混搭）

| 主题 | 气质 | 关键 token |
| --- | --- | --- |
| Graphite | 极简/产品默认 | 冷中性 hue255，强调 `oklch(46% .17 275)` 靛，圆角 6/10/14px，动效 120/200/320ms |
| Porcelain | 温软现代 | 暖中性 hue80，强调 `oklch(60% .13 40)` 陶土，圆角 10/14/20px，动效 140/220/360ms |
| Carbon | 暗色高端 | 暗即底色，表面 14/17.5/21/25%（主题刻意重于通用 6/8/12% 阶梯），强调 `oklch(78% .12 170)` 荧薄荷，无投影（表面阶梯+1px 线），动效 120/180/300ms |
| Signal | 锐利几何 | 零 chroma 中性，强调 `oklch(40% .16 264)`，等宽数字，圆角 2/6/10px，动效 100/160/240ms（快即美学） |

强调色轮换池（连续项目不重复）：靛 `oklch(46% .17 275)` / 青 `oklch(48% .12 195)` / 玫 `oklch(52% .14 15)` / 琥珀 `oklch(58% .14 75)` / 森绿 `oklch(46% .11 155)`。hover ≈ 同色相压暗 5%，浅底色块 ≈ 8-10% 透明度，暗色模式亮度 +15-20%。
仪表盘语义板（L 50-60、C ≤0.15）：success `oklch(60% .12 150)` · warning `oklch(72% .13 75)` · danger `oklch(58% .15 25)` · info `oklch(62% .12 240)`。

营销/品牌色板配方（八组可选起点）：黑+青+哑珊瑚 ｜ 黑+红+奶油+蓝 ｜ 森绿+青柠+雾灰 ｜ 藏青+白+钢青 ｜ 象牙+深蓝+红+金 ｜ 黑+丁香紫 ｜ 黑+琥珀+红 ｜ 炭+白+浅蓝。高端消费品另见 anti-slop.md 的奶油+黄铜禁令与六族轮换。
